## 五、当前仍需关注的环境项

### 1. 外部 realtime gateway
- 当前本地配置下 `agent.realtime.gateway-enabled=false`
- 因此 realtime smoke 只能验证“主链可运行 + 可正确降级”，不能代表真实外部 realtime gateway 已完成环境联通

### 2. RabbitMQ
- Java 启动日志显示 `localhost:5672` 连接拒绝
- 该问题当前不阻塞 `agent/chat` smoke
- 如上线窗口需要相关异步链路，仍需现场验通

### 3. 命令行中文请求编码
- 使用 PowerShell 直接发送中文 JSON 时，建议显式使用 UTF-8 / Unicode
- 否则可能在联调中误把终端编码问题当成写操作识别问题
