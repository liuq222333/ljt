## 一、项目目标

建设一套基于 Agent 的智能检索系统，将以下能力统一到自然语言交互入口：

- 结构化搜索
- 知识检索
- 实时状态确认
- 多轮上下文承接
- 写操作补参数与确认态拦截

当前系统已完成搜索主链、知识主链、实时层与内部正式 realtime gateway，并已完成 Python 优先的 Query Parser / Tool Router / Response Composer / Action Review 四段接入。W12-W14 已完成代码闭环，且 2026-04-12 已完成本地真实 smoke 与一轮本地 Launch rehearsal，后续重点转入真实上线窗口清障与环境操作。
