## 一、项目目标

建设一套基于 Agent 的智能检索系统，将以下能力统一到自然语言交互入口：

- 结构化搜索
- 知识检索
- 实时状态确认
- 多轮上下文承接

当前系统已完成搜索主链、知识主链、实时层骨架，以及 Python 优先的 Query Parser / Tool Router / Response Composer。计划重点已转入 W12，同步推进生产化收尾。
