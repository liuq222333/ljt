#### 12.3 本地活动相关表

##### 12.3.1 local_activity（活动表）
```sql
CREATE TABLE `local_activity` (
  `id` BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '活动ID',
  `title` VARCHAR(200) NOT NULL COMMENT '活动标题',
  `description` TEXT COMMENT '活动描述',
  `category` VARCHAR(50) COMMENT '活动分类',
  `start_time` DATETIME NOT NULL COMMENT '开始时间',
  `end_time` DATETIME COMMENT '结束时间',
  `location` VARCHAR(255) COMMENT '活动地点',
  `latitude` DECIMAL(10,7) COMMENT '纬度',
  `longitude` DECIMAL(10,7) COMMENT '经度',
  `max_participants` INT COMMENT '最大参与人数',
  `current_participants` INT DEFAULT 0 COMMENT '当前参与人数',
  `organizer_id` INT NOT NULL COMMENT '组织者ID',
  `contact_info` VARCHAR(255) COMMENT '联系方式',
  `tags` VARCHAR(500) COMMENT '标签（JSON数组）',
  `status` TINYINT DEFAULT 1 COMMENT '状态：0-已取消，1-进行中，2-已结束',
  `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  `updated_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  INDEX idx_category (`category`),
  INDEX idx_organizer (`organizer_id`),
  INDEX idx_start_time (`start_time`),
  INDEX idx_status (`status`),
  SPATIAL INDEX idx_location (`latitude`, `longitude`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='本地活动表';
```


## 部署指南

### 13.1 环境要求

**服务器环境**:
- 操作系统: Linux (Ubuntu 20.04+ / CentOS 7+) 或 Windows Server
- CPU: 2核心以上
- 内存: 4GB以上
- 硬盘: 50GB以上

**软件依赖**:
- JDK 17
- Maven 3.6+
- Node.js 20.19.0+ 或 22.12.0+
- MySQL 8.0+
- Redis 6.0+
- RabbitMQ 3.9+
- MinIO (最新版)

---

### 13.2 数据库部署

#### 13.2.1 安装MySQL
```bash
# Ubuntu
sudo apt update
sudo apt install mysql-server

# 启动MySQL
sudo systemctl start mysql
sudo systemctl enable mysql
```

#### 13.2.2 创建数据库
```sql
CREATE DATABASE community_service DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

CREATE USER 'community_user'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON community_service.* TO 'community_user'@'localhost';
FLUSH PRIVILEGES;
```

#### 13.2.3 导入数据表
```bash
mysql -u community_user -p community_service < schema.sql
```

---

### 13.3 Redis部署

#### 13.3.1 安装Redis
```bash
# Ubuntu
sudo apt install redis-server

# 启动Redis
sudo systemctl start redis
sudo systemctl enable redis
```

#### 13.3.2 配置Redis
编辑 `/etc/redis/redis.conf`:
```conf
bind 0.0.0.0
port 6379
requirepass your_redis_password
maxmemory 2gb
maxmemory-policy allkeys-lru
```

### 13.4 RabbitMQ部署

#### 13.4.1 安装RabbitMQ
```bash
# Ubuntu
sudo apt install rabbitmq-server

# 启动RabbitMQ
sudo systemctl start rabbitmq-server
sudo systemctl enable rabbitmq-server
```

#### 13.4.2 启用管理插件
```bash
sudo rabbitmq-plugins enable rabbitmq_management

# 创建用户
sudo rabbitmqctl add_user admin your_password
sudo rabbitmqctl set_user_tags admin administrator
sudo rabbitmqctl set_permissions -p / admin ".*" ".*" ".*"
```

访问管理界面: `http://localhost:15672`

---

### 13.5 MinIO部署

#### 13.5.1 安装MinIO
```bash
# 下载MinIO
wget https://dl.min.io/server/minio/release/linux-amd64/minio
chmod +x minio

# 创建数据目录
mkdir -p /data/minio

# 启动MinIO
./minio server /data/minio --console-address ":9001"
```

#### 13.5.2 配置MinIO
- 访问控制台: `http://localhost:9001`
- 默认账号: minioadmin / minioadmin
- 创建bucket: `community-feed`, `community-marketplace`
- 设置访问策略为public（可选）
