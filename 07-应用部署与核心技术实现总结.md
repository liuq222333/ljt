### 13.6 后端部署

#### 13.6.1 配置application.yml
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/community_service?useUnicode=true&characterEncoding=utf8&useSSL=false
    username: community_user
    password: your_password
    driver-class-name: com.mysql.cj.jdbc.Driver

  redis:
    host: localhost
    port: 6379
    password: your_redis_password
    database: 0

  rabbitmq:
    host: localhost
    port: 5672
    username: admin
    password: your_password

minio:
  endpoint: http://localhost:9000
  access-key: admin
  secret-key: admin12345
  bucket: community-feed

server:
  port: 8080
```

#### 13.6.2 打包后端
```bash
cd backend/demo1
mvn clean package -DskipTests

# 生成的jar文件在 target/demo1-0.0.1-SNAPSHOT.jar
```

#### 13.6.3 运行后端
```bash
# 直接运行
java -jar target/demo1-0.0.1-SNAPSHOT.jar

# 后台运行
nohup java -jar target/demo1-0.0.1-SNAPSHOT.jar > app.log 2>&1 &

# 使用systemd管理（推荐）
sudo nano /etc/systemd/system/community-service.service
```

systemd配置:
```ini
[Unit]
Description=Community Service Application
After=network.target

[Service]
Type=simple
User=www-data
WorkingDirectory=/opt/community-service
ExecStart=/usr/bin/java -jar /opt/community-service/demo1.jar
Restart=on-failure

[Install]
WantedBy=multi-user.target
```

启动服务:
```bash
sudo systemctl daemon-reload
sudo systemctl start community-service
sudo systemctl enable community-service
```

#### 13.7.4 Nginx配置
```nginx
server {
    listen 80;
    server_name your-domain.com;

    # 用户端
    location / {
        root /var/www/community-service/user;
        try_files $uri $uri/ /index.html;
    }

    # 管理后台
    location /admin {
        alias /var/www/community-service/admin;
        try_files $uri $uri/ /admin/index.html;
    }

    # API代理
    location /api {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }

    # WebSocket代理
    location /ws {
        proxy_pass http://localhost:8080;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
}
```

部署前端文件:
```bash
sudo mkdir -p /var/www/community-service/user
sudo mkdir -p /var/www/community-service/admin

sudo cp -r front/aaljt/dist/* /var/www/community-service/user/
sudo cp -r front-admin/dist/* /var/www/community-service/admin/

sudo systemctl restart nginx
```

### 14.2 实时通知实现

**RabbitMQ + WebSocket架构**:

1. 发送通知到RabbitMQ队列
2. 消息监听器接收并保存到数据库
3. 通过WebSocket推送给在线用户

```java
// 发送消息
@Autowired
private RabbitTemplate rabbitTemplate;

public void sendNotification(NotificationMessage message) {
    rabbitTemplate.convertAndSend("notification.exchange", "notification.key", message);
}

// 监听消息
@RabbitListener(queues = "notification.queue")
public void handleNotification(NotificationMessage message) {
    // 保存到数据库
    notificationMapper.insert(notification);

    // WebSocket推送
    messagingTemplate.convertAndSendToUser(
        String.valueOf(message.getUserId()),
        "/queue/notifications",
        notification
    );
}
```

### 14.3 图片存储实现

**MinIO对象存储**:

```java
// 上传图片
public String uploadImage(MultipartFile file) {
    String objectKey = "products/" + UUID.randomUUID() + "-" + file.getOriginalFilename();

    minioClient.putObject(
        PutObjectArgs.builder()
            .bucket("community-marketplace")
            .object(objectKey)
            .stream(file.getInputStream(), file.getSize(), -1)
            .contentType(file.getContentType())
            .build()
    );

    return objectKey;
}

// 生成预签名URL
public String getPresignedUrl(String objectKey) {
    return minioClient.getPresignedObjectUrl(
        GetPresignedObjectUrlArgs.builder()
            .bucket("community-marketplace")
            .object(objectKey)
            .method(Method.GET)
            .expiry(7, TimeUnit.DAYS)
            .build()
    );
}
```

## 项目总结

### 15.1 技术亮点

1. **事件驱动架构** - RabbitMQ实现异步消息处理，提高系统响应速度
2. **地理位置服务** - Redis GEO + RediSearch实现高性能附近推荐
3. **实时通信** - WebSocket + RabbitMQ实现实时消息推送
4. **AI智能助手** - 集成大模型提供智能对话和工具调用
5. **对象存储** - MinIO实现图片和文件的分布式存储
6. **微服务架构** - 模块化设计，易于扩展和维护

### 15.2 性能优化

1. **Redis缓存** - 热点数据缓存，减少数据库压力
2. **分页查询** - PageHelper实现高效分页
3. **索引优化** - 数据库索引优化查询性能
4. **异步处理** - RabbitMQ异步处理耗时操作
5. **CDN加速** - 静态资源CDN加速（生产环境）

### 15.3 安全措施

1. **Spring Security** - 统一认证授权
2. **密码加密** - BCrypt加密存储
3. **Token认证** - JWT或自定义Token
4. **CORS配置** - 跨域请求控制
5. **SQL注入防护** - MyBatis参数化查询
6. **XSS防护** - 前端输入过滤

### 15.4 监控运维

**建议添加**:

## 结语

本系统基于事件驱动架构和AI大模型技术，为社区居民提供了一个集成化的服务平台。通过社区集市、本地活动、邻里互助、社区动态等功能模块，实现了社区服务的智能化触达。

系统采用前后端分离架构，使用Spring Boot + Vue3技术栈，集成了Redis、RabbitMQ、MinIO等中间件，具有良好的扩展性和可维护性。

希望本文档能够帮助开发者快速理解和使用本系统。
